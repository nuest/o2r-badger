Extender installation

  1. Install the o2r-extender Chrome extension
    1. Open chrome://extensions/ in Chrome or select "More tools > Extensions" in the menu,
    2. make sure that "Developer mode" in the top-right corner is enabled,
    3. click "Load unpacked extension...", and
    4. select the "extension" directory of o2r-extender from the demo package: badger_demo/o2r-extender/extension
  2. Configure o2r-extender to use local o2r-badger
    1. Open the extender options by right clicking the new "o2r" icon and selecting options. Then change the setting Badge Endpoint to http://localhost/api/1.0/badge/. Click "Save".
    2. Allow mixed content in Chrome when using the extension on one of the supported websites: Select "Load unsafe scripts" by clicking the shield icon in the address bar (detailed explanation).

Demo compendia

This repository comes with demo compendia to to support demonstrations for selected research papers and search terms.

Research websites

Then visit one of the following research websites and search for papers to learn about their badges:

  • Google Scholar (supports result filtering) with example search for "air quality"
  • MS Academic Search with demo paper "Controls on microalgal community structures in cryoconite holes upon high-Arctic glaciers, Svalbard"
  • DOAJ (supports result filtering) with example search for "air quality" and demo paper "A Bayesian posterior predictive framework for weighting ensemble regional climate models"
  • ScienceOpen with demo paper "Underestimation of boreal soil carbon stocks by mathematical soil carbon models linked to soil nutrient status"
  • PLOS
